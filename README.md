# UI Tests

## Тест-кейс по первому пункту чек-листа

FT-01 Работа сортировки по наименованию и стоимости

Предусловие: открыта страница https://automationteststore.com/ и выбрана категория товаров "Apparel & accessories -> T-shirts"

<br>

Шаги выполнения:

1. В dropdown "Sort By:" выбрать метод сортировки "Name A - Z"

    Ожидаемый результат: товары выбранной категории отсортированы по наименованию в алфавитном порядке

2. В dropdown "Sort By:" выбрать метод сортировки "Name Z - A"

    Ожидаемый результат: товары выбранной категории отсортированы по наименованию в обратном алфавитному порядке

3. В dropdown "Sort By:" выбрать метод сортировки "Price Low > High"

    Ожидаемый результат: товары выбранной категории отсортированы по стоимости в порядке возрастания

4. В dropdown "Sort By:" выбрать метод сортировки "Price High > Low"

    Ожидаемый результат: товары выбранной категории отсортированы по стоимости в порядке убывания

<br>

## Тест-кейс по второму пункту чек-листа

FT-02 Работа поиска по наименованию и корзины

Предусловие: открыта страница https://automationteststore.com/

<br>

1. В поле "Search Keywords" ввести "shirt"

    Ожидаемый результат: на странице отображены товары в наименовании которых есть слово "shirt"

2. В dropdown "Sort By:" выбрать метод сортировки "Name A - Z"

    Ожидаемый результат: товары выбранной категории отсортированы по наименованию в алфавитном порядке

3. У второго по счету среди отображенных товаров нажать на изображение

    Ожидаемый результат: открытие страницы с выбором характеристик и количества штук данного товара

4. В поле "Qty" ввести случайное количество товаров 

    Ожидаемый результат: изменение итоговой стоимости позиции при добавлении в корзину

5. Нажать кнопку "Add to Cart" 

    Ожидаемый результат: открытие страницы с корзиной, появление в корзине выбранного товара в указанном количестве

6. Повторить шаги 1, 2

    Ожидаемый результат: товары в наименовании которых есть слово "shirt" отсортированы по наименованию в алфавитном порядке

7. Повторить шаги 3 - 5 для третьего по счету товара среди отображенных

    Ожидаемый результат: открытие страницы с корзиной, появление в корзине выбранного товара в указанном количестве вместе с предыдущим

8. Определить наименьший по стоимости товар в корзине и удвоить его количество

    Ожидаемый результат: количество товара с наименьшей стоимостью изменилось на вдвое бьльшее, итоговая стоимость данного товара выросла вдвое

9. Проверить итоговую стоимость заказа

    Ожидаемый результат: стоимость заказа равна - (Unit Price 1 * Quantity 1) + (Unit Price 2 * Quantity 2)

## Тест-кейс по третьему пункту чек-листа

FT-03 Работа корзины

Предусловие: открыта страница https://automationteststore.com/

<br>

1. На главной странице выбрать случайный товар и нажать на изображение его карточки

    Ожидаемый результат: открытие страницы с выбором характеристик и количества штук данного товара

2. В поле "Qty" ввести случайное количество товаров 

    Ожидаемый результат: изменение итоговой стоимости позиции при добавлении в корзину

3. Нажать кнопку "Add to Cart" 

    Ожидаемый результат: открытие страницы с корзиной, появление в корзине выбранного товара в указанном количестве

4. Нажать кнопку "Home"

    Ожидаемый результат: открытие главной страницы сайта

5. Повторить шаги 1 - 4 четыре раза

    Ожидаемый результат: в корзине находятся 5 позиций товаров со случайным количеством

6. В корзине определить товары, которые по порядку являются четными, удалить их из корзины

    Ожидаемый результат: товары, которые по порядку являлись четными удалены из корзины

7. Проверить итоговую стоимость заказа

    Ожидаемый результат: стоимость заказа равна сумме произведений стоимости и количество у оставшихся товаров
